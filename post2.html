Building My First Website

Published: August 1, 2025

This is the post where I share how I built this personal blog using basic web technologies: HTML, CSS, and JavaScript. It’s simple, fast, and easy to edit — perfect for learning and sharing.

🌐 Tools I Used

  • VS Code for coding
  • Google Chrome for testing
  • HTML/CSS for structure and styling
  • JavaScript for interactivity

✨ Features I Added

  • Scroll animations (triggered when sections come into view)
  • Dark mode toggle (for eyes and vibes 😎)
  • “Back to top” button (for easy navigation)

This site is mobile-friendly and built with clean code. It’s a perfect starter project and I'm proud of it. I’ll keep improving and adding new sections as I learn more. 💪

💡 What’s Next?

I want to integrate a CMS or backend later, maybe using PHP or Firebase. Also, I might add AI features in future, like chatbots or blog content generation tools.
