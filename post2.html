Building My First Website

August 1, 2025 7 min read

Building this blog has been an incredible learning experience. I started with basic HTML and CSS, then added JavaScript for interactivity like dark mode and scroll animations.

Here's my approach:

  • Start with a clean HTML structure
  • Use CSS variables for theming (light/dark mode)
  • Implement JavaScript for interactive elements
  • Make it responsive for all devices
  • Optimize for performance and accessibility

Here's a simple example of the dark mode toggle JavaScript:

// Check for saved theme preference or respect OS preference
if (localStorage.getItem('theme') === 'dark' ||
(window.matchMedia('(prefers-color-scheme: dark)').matches
&& !localStorage.getItem('theme'))) {
  document.documentElement.setAttribute('data-theme', 'dark');
} else {
  document.documentElement.setAttribute('data-theme', 'light');
}

Tip: Use CSS variables for colors to make theme switching easier!
